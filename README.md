# Метрика — Студия дизайна интерьера

Лендинг для студии дизайна интерьера **Метрика**. Одностраничный сайт с адаптивной вёрсткой, галереей проектов и формой обратной связи.

## 🖥️ Демо

> Разверни проект локально или открой в GitHub Pages - https://artemdreamchik.github.io/metrika/

## 📌 Разделы страницы

- **О нас** — краткое описание студии
- **Проекты** — карточки реализованных проектов
- **Услуги** — прайс на планировочное решение, дизайн-проект, авторский надзор и комплектацию
- **Отзывы** — карточки с отзывами клиентов
- **Галерея** — слайдер с фотографиями проектов
- **Контакты** — телефон и e-mail

## ⚙️ Функциональность

- **Burger-меню** — адаптивная навигация для мобильных устройств
- **Слайдер галереи** — переключение между проектами стрелками
- **Popup «Заказать услугу»** — модальное окно с формой (имя, email, телефон)

## 🛠️ Стек

| Технология | Описание |
|---|---|
| HTML5 | Разметка страницы |
| CSS3 | Стили, адаптивность |
| JavaScript (ES6+) | Интерактивность (модули) |
| [Manrope](https://fonts.google.com/specimen/Manrope) | Шрифт (Google Fonts) |

Без сборщиков и сторонних зависимостей — проект запускается напрямую в браузере.

## 📁 Структура проекта

```
metrika/
├── index.html          # Основная страница
├── burger.js           # Логика мобильного меню
├── gallery.js          # Слайдер галереи
├── popup.js            # Модальное окно
├── styles/
│   ├── reset.css       # Сброс стилей
│   └── style.css       # Основные стили
└── resorces/           # Изображения и иконки
```

## 🚀 Запуск

Никакой установки не требуется. Просто открой `index.html` в браузере:

```bash
# Клонируй репозиторий
git clone https://github.com/ArtemDreamchik/metrika.git

# Перейди в папку проекта
cd metrika

# Открой index.html в браузере
# (или запусти локальный сервер, например через VS Code Live Server)
```

> **Рекомендуется** открывать через локальный сервер (например, расширение [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) в VS Code), чтобы ES-модули (`type="module"`) работали корректно.
